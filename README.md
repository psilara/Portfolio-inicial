# Portfolio de Pablo Nicolás Siles Arano

Portfolio personal responsive creado en HTML, CSS y JavaScript con Vite.

## Cómo ejecutarlo

```bash
cd pablo-portfolio
npm install
npm run dev
```

Después abre en el navegador la URL que aparezca en la terminal, normalmente:

```bash
http://localhost:5173
```

## Crear versión para publicar

```bash
npm run build
```

La web final quedará en la carpeta `dist`.

## Editar contenido

- Textos, experiencia, formación y proyectos: `index.html`
- Colores, diseño y responsive: `styles.css`
- Menú móvil, animaciones y botón de copiar email: `script.js`
- Foto y CV: carpeta `assets/`
